このページについて

このサイトでは、下記のデータをSPARQLエンドポイントに登録して公開しています。

各エンドポイントにSPARQLクエリを送信する場合は、エンドポイントのURLに送信してください。

ブラウザ上でSPARQLクエリの検索結果を確認したい場合は、本ページ下部にあるクエリ入力・実行フォームを利用してください。

SPARQLエンドポイントの一覧

ISIL-LOD（図書館及び関連組織のための国際標準識別子）試行版
入手元: 国立国会図書館 オープンデータセット
SPARQLエンドポイントのURL: https://lib-lod.tsunagun.org/sparql/isil-lod-jp/query
NDLSH（国立国会図書館件名標目表）
入手元: Web NDL Authorities 一括ダウンロード用ファイル
SPARQLエンドポイントのURL: https://lib-lod.tsunagun.org/sparql/ndlsh/query
NDC（日本十進分類法）第9版
入手元: NDC（日本十進分類法）
SPARQLエンドポイントのURL: https://lib-lod.tsunagun.org/sparql/ndc9/query
NDC（日本十進分類法）第8版
入手元: NDC（日本十進分類法）
SPARQLエンドポイントのURL: https://lib-lod.tsunagun.org/sparql/ndc8/query

SPARQLクエリを使ってみる

デフォルトではISIL-LODのエンドポイントに接続されます。

接続先を変える場合は、SPARQLクエリ入力部の上部にある入力ボックスに、接続したいエンドポイントのURLを入力してください。
